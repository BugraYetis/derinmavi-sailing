# ⛵ Derinmavi — Yelken Okulu

> 30 yıllık denizcilik tutkusuyla Demre/Antalya merkezli yelken okulu kurumsal sitesi

## 🌐 Demo
**[Canlı Demo](https://derinmavi.com.tr/)**

## ✨ Özellikler
- Denizcilik temalı UI tasarımı (sinyal bayrakları navigasyonu)
- İnteraktif harita ile gemi konumu gösterimi
- GPS koordinat ve AIS veri entegrasyonu
- Gerçek zamanlı gemi bilgileri (hız, rota, derinlik)
- Eğitim programları ve rota bilgileri
- Sosyal medya entegrasyonu (Instagram, WhatsApp, YouTube)
- Responsive mobil tasarım

## 🛠️ Teknolojiler
`HTML5` `CSS3` `JavaScript` `Mapping Library` `GPS/AIS Integration` `Responsive Design`

## 📊 İstatistikler
- 3.000+ eğitim alan öğrenci
- 50.000+ deniz mili deneyim
- 30 yıllık denizcilik tutkusu
